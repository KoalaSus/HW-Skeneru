Zde jsou součástky, které tvoří držáky pro motor a lasery.

Modely jsou přispůsobeny pro šroubky velikosti 4,5mm a podložky 9-11mm, tady záleží na velikosti, protože s podložkou 9 mm se dá ještě přizpůsobovat.

Držák pro motor je vymodelovaný, jako jeho obal s rezervou 0,25mm do výšky 33mm.

Lasery jsou velikostně stejný a to do bodu středu 50 mm.
